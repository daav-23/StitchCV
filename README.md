# StitchCV

> CV interactif open source conçu avec HTML5, CSS3, JavaScript & Tailwind CSS.

## 🌐 Aperçu

**StitchCV** est un portfolio web moderne mettant en avant une approche **Mobile-First** et une identité visuelle soignée basée sur les principes du **Material Design 3**.

- ✅ Design responsive (mobile, tablette, desktop)
- ✅ Mode sombre / clair automatique
- ✅ Impression optimisée (styles print dédiés)
- ✅ Chatbot interactif avec Q&A

## 🛠️ Stack technique

| Technologie | Usage |
|-------------|-------|
| HTML5 | Structure sémantique |
| CSS3 + Tailwind CSS | Styling & utilitaires |
| JavaScript vanilla | Interactivité (dark mode, chatbot, modal) |
| Google Fonts (Manrope) | Typographie |
| Material Symbols | Icônes |
